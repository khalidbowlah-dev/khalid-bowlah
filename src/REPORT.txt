JAVA FOOD ORDERING SYSTEM - PROJECT REPORT

Student Name: Khaled Bowllah Moussa
Project: Console-based Food Ordering System

1. OVERVIEW
   This project is a restaurant simulation where users can view a menu,
   add items to a cart, and process payments.

2. OOP CONCEPTS
   - Encapsulation: Used in 'MenuItem' (private fields).
   - Inheritance: 'Customer' extends 'User'.
   - Polymorphism: 'Payment' system handles Cash vs Credit Card differently.
   - Interface: 'MenuItem' implements 'Orderable'.

3. TESTING
   - JUnit 5 was used to verify price storage and payment logic.

4. CONCLUSION
   The project successfully demonstrates core Java OOP principles.
